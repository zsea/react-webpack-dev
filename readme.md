一个自用的webpack+react的开发环境。

支持以下功能：

* 热加载
* less
* react
* ES6
* sass
* postcss

### loaders

*  babel-loader
*  css-loader
*  file-loader
*   json-loader
*   less-loader
*   postcss-loader
*   raw-loader
*   sass-loader
*   style-loader
*   url-loader

### 目录说明

* src:源代码所在目录
* templates:生成的html代码所在目录
* dist:目录代码所在目录

### 文件说明

* src/index.js：项目入口文件
* templates/index.ejs：项目主页面html模版。

### 命令

* npm run dev：启动开发环境，可以在浏览器中查看效果。<http://127.0.0.1:3000>
* npm run build：生成目标代码
